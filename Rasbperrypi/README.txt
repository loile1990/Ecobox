Raspberry Pi Configuration Script

This file contains a script to configure the Raspberry Pi and create the monitoring and capture services, as well as the UDP server.

Follow the instructions below to use it:

Connect to the Raspberry Pi without knowing the IP address assigned by DHCP using the following command:

ssh hownet-ecobox-1@raspberrypi.local
Password: hownet123!

SSID: ecobox
Password: ecobox123

This script is already deployed on the Raspberry Pi.
You just need to plug it in and start collecting data.
